# cf-ddns

## usage

```
> cf-ddns -h
usage: cf-ddns [-6] -t <API_TOKEN> -d <DOMAIN>

optional arguments:
  -h, --help      show this help message and exit
  -6              enable IPv6 mode
  -t <API_TOKEN>  your cf api token
  -d <DOMAIN>     your cf domain name


```
